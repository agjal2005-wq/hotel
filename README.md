# Hotel Website

A responsive hotel website built using **HTML5** and **CSS3**.  
This project focuses on clean UI design, layout structuring, and basic responsiveness.

## Live Demo
https://agjal2005-wq.github.io/hotel/

## Features
- Clean and simple user interface
- Responsive layout using CSS
- Sections for hotel categories (Business, Luxury, Resort)
- Image-based design

## Technologies Used
- HTML5
- CSS3
